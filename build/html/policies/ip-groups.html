2. IP guruhlar (IP Groups)

IP guruhlar bir nechta IP manzil yoki tarmoqni bitta nom ostida birlashtirish imkonini beradi. So’ngra bu nomdan firewall qoidalarida foydalanish mumkin — bu qoidalarni soddalashtiradi va boshqarishni osonlashtiradi. Bu bo’limga o’tish uchun chap menyudagi Firewall → IP Groups bo’limini tanlang (/ip-groups).

2.1. Nima uchun kerak

Tasavvur qiling, sizda 10 ta server bor va ularning hammasiga bir xil ruxsat qoidasi kerak. Har bir server uchun alohida qoida yozish o’rniga, ularni bitta IP guruhga (Serverlar) birlashtirib, bitta qoida yozasiz. Keyinchalik yangi server qo’shilganda — faqat guruhga manzil qo’shasiz, qoidalarni o’zgartirish shart emas.

2.2. IP guruh yaratish

  1. Guruh qo’shish (Add Group) tugmasini bosing.

  2. Quyidagilarni kiriting:

    • Nom (Name) — guruhning tushunarli nomi (masalan, Veb-serverlar).

    • Manzillar — bitta yoki bir nechta IP manzil yoki tarmoq, CIDR ko’rinishida (masalan, 192.168.10.20, 192.168.10.0/24).

  3. Saqlash (Save) tugmasini bosing.

2.3. Guruhdan foydalanish

IP guruh yaratilgandan so’ng, firewall qoidasi (Firewall qoidalari (Firewall Policies / ACL)) yaratishda Manba yoki Manzil maydonida alohida IP o’rniga guruh nomini tanlashingiz mumkin.

2.4. Guruhni tahrirlash va o’chirish

  • Tahrirlash — guruhga manzil qo’shish yoki olib tashlash.

  • O’chirish — guruhni o’chirish.

Note

Agar IP guruh biror firewall qoidasida ishlatilayotgan bo’lsa, uni o’chirishdan oldin shu qoidalarni yangilang yoki o’chiring.
